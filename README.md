# 东坡清风纪

> 苏轼一生互动叙事游戏

**项目定位**: 移动端H5互动叙事游戏，玩家作为"东坡遗卷修复人"，修复苏轼人生中的关键残页。

## 游戏特色

- 电影式开场：墨滴扩散、宣纸显影、水墨意境
- 互动叙事：玩家选择影响变量、台词、画面和结局
- 六大账目系统：清风账、风骨账、民生账、烟火账、亲友账、名声账
- 史实与改编标注：每处关键场景配有史实卡和改编说明
- 5种结局分支：根据玩家选择触发不同结局

## 开发

```bash
npm install
npm run dev      # 开发模式
npm run build    # 构建生产版本
```

## 技术栈

- Vite 5 + Vue 3
- JSON剧情数据
- CSS动画
- 移动端优先设计

## 内容范围（MVP）

MVP只做到黄州阶段，包含：序幕、第一至三幕、临时结果页、分享海报、史实说明。

## 设计原则

- 不虚构苏轼贪腐情节
- 不把改编当史实
- 廉洁主题自然，不生硬说教
- 视觉：冷青灰、水墨、宣纸、朱砂点睛

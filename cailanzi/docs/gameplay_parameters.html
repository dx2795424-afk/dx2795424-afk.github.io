关于"寒潮天露地青菜几乎无法收获"的边界：

真实情况：低温可能影响露地蔬菜生长和收获，但不代表所有露地蔬菜在寒潮下绝收。设施农业文件（A03）支持"设施农业有助于保障农产品稳定安全供给"，但不能直接证明"寒潮天露地青菜几乎无法收获"是普遍现实结论。

因此 v2 改为 C12（无法正常收获）+ C13（恢复可收获状态）的定性差值，不再使用 0% / 100% / ×2 / 20% / 8% 等易误导的精确百分比。

常见疑问：寒潮天真的一片青菜也收不到吗？

答：低温可能影响露地蔬菜生长和收获；游戏为形成 5 日挑战中的明确差值，将其简化为"寒潮状态下，露地青菜无法正常收获"（C12）。该结果不代表现实中所有露地蔬菜在寒潮下绝收。启用温室后恢复可收获状态（C13），体现设施农业的机制价值（A03）。

六、防御性问答

Q1：为什么不写精确数字（30% / 12% / 15%）？

A1：当前公开资料不足以为不同地区、品种、生产条件提供统一固定系数。本作品不制造虚假精确性，改用方向性表述 + 显性标注的游戏化模拟参数。

Q2：游戏化模拟数值和现实数据会不会矛盾？

A2：不会。

  • A 类政策（事实背景）：不嵌入精确数字。
  • B 类机制（方向转译）：只写"有助于 / 提升 / 降低"，不附具体数字。
  • C 类游戏化模拟（差值设计）：必须显性标注"游戏化模拟数值，不是现实统计复刻"。

Q3：玩家会不会误以为这些数字是真实的？

A3：作品通过四道防线降低误读：

  1. 笔记本 / 最终报告中显性标注"以上数值为游戏化模拟"
  2. 数据说明页（./fact_check.html）展示 A/B/C 三层分类和未登记清单
  3. 早间播报（Day 2/3/4）显示具体 C 类数值时附"游戏化模拟"标识
  4. 需要核验资料时，可查看 总来源登记表

Q4：作品是新闻游戏还是纯娱乐？

A4：互动新闻游戏。

  • 真实部分：政策背景（A 类）+ 机制方向（B 类）—— 来自国务院 / 各部委公开文件
  • 模拟部分：游戏化数值（C 类）—— 显性标注，让玩家区分
  • 创新部分：让玩家通过 5 日个人生存挑战，体验"未修科技 vs 修科技"的差异

Q5：为什么不加更多科技（如智能灌溉、无人农场）？

A5：作品聚焦"一顿饭的全链路"，选择 3 个核心科技（冷链 / 温室 / 智慧鸡舍）作为主线：

  • 冷链：对应"流通环节"
  • 温室：对应"生产环节"（设施农业）
  • 智慧鸡舍：对应"养殖环节"

这 3 个环节足以体现"科技 → 食物 → 餐桌"的全链路，避免堆砌导致游戏机制复杂。 辅助科技（增氧机 C09-C11、水肥一体化 B05、价格监测大屏 C08）作为可选扩展。

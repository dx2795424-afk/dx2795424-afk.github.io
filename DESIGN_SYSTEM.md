# 设计系统

## 色彩系统

### 主色
- 墨黑: `#1a1a1a`
- 青灰: `#3d4f5f`
- 月白: `#e8e4df`
- 宣纸米: `#f5f0e6`

### 辅色
- 旧金: `#b8956c`
- 浅赭: `#c4a77d`
- 竹青: `#5c7a5c`

### 点缀色（限用）
- 朱砂: `#8b3a3a`（仅用于印章、危机提示）

## 分幕色彩

| 场景 | 色彩 |
|------|------|
| 眉山 | 宣纸米、雨灰、暖灯一点点 |
| 京城 | 青灰、墨黑、冷白 |
| 乌台 | 黑、灰、暗朱砂 |
| 黄州 | 土黄、灰青、小灶低饱和暖色 |
| 承天寺 | 月白、蓝灰、竹影黑 |
| 结果页 | 冷青灰+朱砂印章 |

## 字体

- 标题: `Noto Serif SC` (思源宋体)
- 正文: `Noto Sans SC` (思源黑体)
- 备选: 系统默认衬线/无衬线字体

## 间距系统

- xs: 0.25rem
- sm: 0.5rem
- md: 1rem
- lg: 1.5rem
- xl: 2rem
- 2xl: 3rem

## 动画原则

- 慢
- 克制
- 电影感
- 不花哨
- 不堆特效
- 不影响手机性能

### 动画时长
- fast: 200ms
- normal: 400ms
- slow: 800ms

### 缓动函数
- `cubic-bezier(0.16, 1, 0.3, 1)` - ease-out
- `cubic-bezier(0.65, 0, 0.35, 1)` - ease-in-out

## 禁止风格

- 大红廉政展板
- 喜庆国潮
- 低幼Q版
- 纯表情包
- PPT风
- 答题小程序模板感
- AI味很重的古风堆砌

## 组件规范

### 按钮
- 高度: 48px (移动端友好)
- 圆角: 4px
- 内边距: 1rem 2rem

### 卡片
- 背景: rgba(26, 26, 26, 0.7)
- 边框: 1px solid rgba(232, 228, 223, 0.1)
- 圆角: 8px
- 内边距: 1rem

### 史实卡
- 左边框: 3px solid #5c7a5c
- 背景: rgba(92, 122, 92, 0.1)

### 改编卡
- 左边框: 3px solid #b8956c
- 背景: rgba(184, 149, 108, 0.1)

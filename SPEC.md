# 东坡清风纪 - 产品规格说明书

## 1. 项目概述

**项目名称**: 东坡清风纪
**副标题**: 苏轼一生互动叙事游戏
**传播标题**: 今天谁在捞东坡？
**主题句**: 一生风雨，一本清账。

**核心定位**: 移动端H5互动叙事游戏，玩家作为"东坡遗卷修复人"，修复苏轼人生中的关键残页。游戏涵盖眉山少年、京城成名、乌台风暴、黄州贬谪、承天寺夜游等关键节点。

**目标用户**: 校园用户，适合扫码传播
**技术栈**: Vite + Vue 3 + JSON剧情数据 + GSAP动画

---

## 2. 变量系统

初始值均为50，范围0-100：

| 变量 | 游戏名 | 含义 |
|------|--------|------|
| qingfeng | 清风账 | 公私边界、拒礼、守规则 |
| fenggu | 风骨账 | 风波和困境中是否守住人格 |
| minsheng | 民生账 | 是否关心百姓和公共责任 |
| yanhuo | 烟火账 | 是否能在困境中真实生活 |
| qinyou | 亲友账 | 亲友承担的代价 |
| mingsheng | 名声账 | 才名和传播风险 |

## 4. 视觉设计

### 色彩系统
- 主色: 墨黑 #1a1a1a, 青灰 #3d4f5f, 月白 #e8e4df, 宣纸米 #f5f0e6

| 场景 | 色彩 |

| 眉山 | 宣纸米、雨灰、暖灯一点点 |

| 黄州 | 土黄、灰青、小灶低饱和暖色 |
| 承天寺 | 月白、蓝灰、竹影黑 |

## 7. 验收标准

### 内容验收
- [ ] 不是苏轼百科
- [ ] 不是普通廉洁教育问答
- [ ] 有电影式开场
- [ ] 有苏轼、苏辙、村民、张怀民
- [ ] 有乌台风暴、黄州、猪肉颂、承天寺夜游
- [ ] 有史实卡和改编卡
- [ ] 不把二创当史实
- [ ] 不虚构苏轼贪腐情节
- [ ] 廉洁主题自然，不生硬说教
- [ ] 正经中有冷幽默

### 交互验收
- [ ] 不是滑动H5
- [ ] 有点击探索
- [ ] 有选择分支
- [ ] 有变量变化
- [ ] 有后续台词差异
- [ ] 有临时结果页
- [ ] 有分享海报
- [ ] 有黄州猪肉预算/人情判断

### 视觉验收
- [ ] 整体冷青灰、水墨、宣纸、朱砂点睛
- [ ] 不大面积使用红色
- [ ] 不像廉政展板
- [ ] 有凄凉英雄感
- [ ] 有宋代生活美学

### 技术验收
- [ ] 手机端优先
- [ ] 微信浏览器可打开
- [ ] 页面加载流畅
- [ ] 变量正确记录
- [ ] 分享海报能生成
